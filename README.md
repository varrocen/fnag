# Mower

Demo project for FNAG.

[![Build Status](https://travis-ci.org/varrocen/fnag.svg?branch=master)](https://travis-ci.org/varrocen/fnag)

Requirements
------------

* JDK 1.9 or later
* Maven 3.5+

Run the application
-------------------

To run the application, execute:

    mvn spring-boot:run
